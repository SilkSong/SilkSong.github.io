Hello World

2017-07-11

这是我的第一个个人博客，断断续续花费了大约10个小时的时间，从购买域名，到建立 GitHub Pages 仓库，然后 DSN 解析，终于在这个纷繁的网络中有了属于自己的一块安静的归属地。然后试着看官方文档使用 Jekyll 搭建博客，无奈迷失在配置 Ruby 等各种各样的命令行设置里，实在摸不着头脑，转而决定采用更加简单的基于 Node 的 Hexo，过程轻松了一些，但也是踩了无数的坑，最终选择了 maupassant 这个看起来干净，简洁的主题，让这个博客成功跑了起来，我现在正在打下它的第一篇博客，“Hello World!”
虽然还有很多的个性化配置尚未搞清楚，但当下还是先专注在博客的内容而不是形式吧，等它丰富一些，再把它弄的好看一点…
